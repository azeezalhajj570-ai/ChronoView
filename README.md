# ChronoView

ChronoView is a native Android premium watch store app built with Kotlin + Jetpack Compose.

## Features
- Watch catalog in a responsive grid layout.
- Search by watch name or brand.
- Category filters (Luxury, Sport, Casual).
- Product detail screen with image carousel.
- Local AI-style description generator (template-based, on-device).
- Shopping cart with quantity controls.
- Total price calculation and UI-only checkout action.
- Local cart persistence using Room Database.

## Tech Stack
- Kotlin
- Jetpack Compose + Material 3
- Navigation Compose
- Room Database
- Coil (image loading)

## Run
1. Open the project in Android Studio (Giraffe+ recommended).
2. Let Gradle sync.
3. Run the `app` configuration on an emulator or physical Android device.
